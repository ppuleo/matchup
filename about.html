About Travelodge Match-Up

Travelodge Match-Up: a memory matching game about everyone's favorite motel chain.

When I moved to San Francisco a few months ago, I stayed in a Travelodge for a few weeks while I looked for a reasonable apartment in the city (Pro tip: there's no such thing). In honor of that stay, I decided to make a memory matching game commemorating all of my least favorite things about Travelodge. Hooray for cheap motels!

Travelodge Match-Up is built on bootstrap and uses some fancy new CSS3 transforms and transitions to flip the cards. Also, you can cheat by clicking extra fast.
